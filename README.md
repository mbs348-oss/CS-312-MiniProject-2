# SipScout

SipScout is a simple cocktail recipe finder website. It was built with Node.js, Express.js, EJS, Axios, and TheCocktailDB API.

## What This Project Does

This website lets a user pick a drink category and look through cocktail recipes. When the user chooses a category, the app gets matching drinks from TheCocktailDB and shows them on the page.

The recipes and pictures are not saved inside this project. They come from TheCocktailDB through an API request.

## Features

- Choose a drink category
- View matching cocktail results
- Click a drink to see the full recipe
- See ingredients, measurements, instructions, and pictures
- Use a random drink option
- Works on desktop and smaller screens

## Tools Used

- Node.js
- Express.js
- EJS
- Axios
- CSS
- TheCocktailDB API
